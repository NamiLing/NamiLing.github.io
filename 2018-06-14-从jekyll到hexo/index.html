Jekyll还是怎么都觉得不好用，换成了Hexo，主题用的最大众的NexT。

首先要明白博客的定位。既然是博客，就是给人看的，随便的笔记就不要发上去了。因此也不要在意静态博客系统不灵活这个事。

参考文档：

  • Hexo
  • Next

有什么问题看基本上面两个都能解决。

安装node.js和建立站点

我是mac，用brew安装就好

1
2

$ brew install node
$ npm install -g hexo-cli

建立站点

1
2
3

$ hexo init <folder>
$ cd <folder>
$ npm install

打开hexo服务后，就可以看生成的站点了

1

$ hexo s

在侧边栏显示分类

在NexT主题的layout/_custom/sidebar.swig里添加：

1
2
3
4
5

<div class="category-all-page">
		<div class="category-all">
		  {{ list_categories({depth: 1}) }}
		</div>
</div>

标签同理，使用 list_tags 函数即可。
